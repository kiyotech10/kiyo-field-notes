# 動画スクリプト：「境界知能は7人に1人」は、統計が先に決めた数字だ

**想定尺**：4〜5分  
**対象**：統計・社会問題に興味がある20〜40代  
**対応記事**：`/lifehack/borderline-intelligence-statistics`

---

```

【ナレーション】
「境界知能は、7人に1人いる。」

最近SNSやニュースでよく見る数字だ。

でも、ちょっと待ってほしい。

この数字、どこから来たか知ってる？

【テロップ】

```

## 🎬 SCENE 2｜境界知能とは（0:20〜1:00）

【テロップ】
```
境界知能とは？
IQ 70〜85 のグレーゾーン
```

【ナレーション】
境界知能というのは、IQがおよそ70から85の範囲にある状態のことだ。

知的障害はIQ70未満。平均的な知能はIQ85以上。

その「あいだ」に位置する、グレーゾーンと呼ばれる領域。

【テロップ】
```
知的障害　　　平均
IQ 70未満 ← ここ → IQ 85以上
　　　　　境界知能
　　　　（IQ 70〜85）
```

【ナレーション】
で、IQというのは、平均を100、標準偏差を15として設計されている。

正規分布——あの釣り鐘型のグラフに従う。

IQ 70〜85の範囲には、数学的に**全体の約13.6%**が収まる。

7人に1人は、この13.6%から来ている。

【テロップ】
```
13.6% ≈ 7人に1人
```

## 🎬 SCENE 3｜核心の問い（1:00〜1:50）

【テロップ】
```
ここで一つ
問いたい
```

【ナレーション】
「7人に1人」という数字は、

現実の中に存在していたものを——**発見した**数字なのか。

それとも、カテゴリーを定義した結果として——**生まれた**数字なのか。

【テロップ】
```
「発見」 or 「定義の結果」
どっち？
```

【ナレーション】
答えは、**後者**だ。

IQを正規分布で設計した時点で、平均から1〜2標準偏差下に収まる人の割合は、**数学的にもう決まっていた**。

「境界知能」という名前を、その範囲にあとからつけた瞬間——

7人に1人という数字は、**自動的に確定した**。

【テロップ】
```
名前をつけた瞬間
数字は確定していた
```

## 🎬 SCENE 4｜統計の作られ方（1:50〜2:50）

【テロップ】
```
統計カテゴリーの構造
```

【ナレーション】
統計的なカテゴリーには、こういう順番がある。

【テロップ・順番に表示】
```
① どこかに線を引く（定義する）
② 線の内側の人を数える
③「○人に1人がXだ」と言う
```

【ナレーション】
「発見→数値」じゃなくて、「定義→数値」なんだ。

そして、これは境界知能だけの話じゃない。

【テロップ】
```
他にも同じ構造：

・高血圧の基準値
・貧困線
・肥満（BMI）の定義
```

【ナレーション】
高血圧の基準は130なのか、140なのか。

貧困線はどこで引くのか。

BMIで25を超えたら肥満、というのはなぜ25なのか。

全部、**先に線を引いて、後から数えている**。

線の位置が変われば——有病率も変わる。

【テロップ】
```
線の位置が変われば
数字も変わる
```

## 🎬 SCENE 5｜数字の重さと落とし穴（2:50〜3:40）

【ナレーション】
「7人に1人」という数字は、インパクトがある。

それだけ多くの人が困難を抱えている——という認識につながって、支援を呼びかける文脈でよく使われる。

その動機を否定したいわけじゃない。

【テロップ】
```
支援の必要性を
訴えること自体は大事
```

【ナレーション】
ただ、数字の作られ方を知らないまま受け取ると——

「数字が現実を映した鏡だ」と思い込んでしまう。

【テロップ】
```
数字 ≠ 現実をそのまま映した鏡
```

【ナレーション】
境界知能のIQ85という上限は、なぜ85なのか。

なぜ80じゃなくて、90でもなくて——85なのか。

その根拠を問えるかどうかが、統計リテラシーの核心にある。
